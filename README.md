# Dots

dots is a note taking app that runs heavily on the browser. 

## Features

- Markdown Editor
- Installable
- Lightweight

## Getting Started

1. Clone the repository
```
git clone https://github.com/jalusw/dots.git
```

2. Serve with your own web server eg. serve
```
npx serve .
```
